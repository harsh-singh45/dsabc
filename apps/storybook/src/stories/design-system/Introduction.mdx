# Intelation Design System

Welcome to the Intelation Design System component library. This is an enterprise-grade design system providing consistent UI components, design tokens, and developer tools for building scalable applications efficiently.

**Version:** 1.0.0 | **Status:** Production Ready

---

## Getting Started

### Installation

Install the design system packages in your project:

```bash
npm install @intelation/ui @intelation/tokens
```

### Basic Usage

```tsx
import { Button, Header, Input } from '@intelation/ui';
import '@intelation/ui/styles';

function App() {
  return (
    <>
      <Header 
        logo={<Logo />} 
        navigationItems={[
          { label: 'Home', href: '/' },
          { label: 'About', href: '/about' }
        ]} 
      />
      <Button 
        variant="solid" 
        colorScheme="primary" 
        size="md"
      >
        Click me
      </Button>
      <Input 
        placeholder="Enter text..." 
        label="Username" 
      />
    </>
  );
}
```

### Tailwind CSS Setup

Configure Tailwind to use the design tokens:

```javascript
// tailwind.config.js
import tokensConfig from '@intelation/tokens/tailwind';

export default {
  content: ['./src/**/*.{js,jsx,ts,tsx}'],
  presets: [tokensConfig],
};
```

## What's Included

### Packages

**@intelation/tokens**
- Design tokens (colors, spacing, typography, etc.)
- Available in CSS, SCSS, JavaScript, TypeScript, and Tailwind formats
- Automatic token resolution and transformation

**@intelation/ui**
- React component library built with TypeScript and Tailwind CSS
- Accessible components using React Aria
- Fully typed with comprehensive props and variants

### Technology Stack

- **React 18** - Component library framework
- **TypeScript 5.2** - Type-safe development
- **Tailwind CSS 4** - Utility-first styling
- **React Aria 3.33** - Accessible component primitives
- **Turborepo** - Monorepo orchestration
